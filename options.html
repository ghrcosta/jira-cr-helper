Jira CR Helper
Browser extension to simplify creating Jira CRs with default options.

How to use
This extension helps create Jira CRs with some default values. See more information here.
Notes:
  • Jira base URL and CR types are hardcoded.
  • Components' name must match the name in Jira (case sensitive).
  • You can set a default description for each component, but note that "Bug" and "Defect" CRs, due to their nature, won't have the default description.
