Jira CR Helper
Simplifies creating Jira CRs with default values


How to use
This extension works by generating a direct link to Jira's "Create issue" page, which allows defining some values via query parameters. See more information here.

Jira base URL, Project ID and CR types are all hardcoded since they almost never change. You can also add a default description, but it's optional. Note that "Bug", "Defect" and "Epic" CRs, due to their nature, won't have the default description.
